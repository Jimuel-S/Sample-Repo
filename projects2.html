Cass Mago

https://cassmago.com

Visit Site

Purpose

This was given to us as a group task consisting of four members, with work individually distributed. It serves as our Prelim Output.

Features

The Cassmago website serves as a promotional platform showcasing the company’s various projects. It highlights the company’s expertise, and completed works. The website includes a project gallery with high-quality images, and an about page that provides insights into Cassmago’s mission and values. To enhance user experience, the site features intuitive navigation, responsive design, and optimized loading speed, ensuring seamless browsing across all devices.

Tools/Technologies Used

We used WordPress to create the website and published it using Hostinger. For images, we used Photoshop. For team communication, we utilized GitHub and Discord.
