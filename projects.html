Purpose

This website was created as part of an academic requirement for our Web Development course. Built using WordPress, it serves as a practical application of the skills learned in designing, customizing, and managing a website.

Features

This website serves as an informational platform where users can explore interesting facts about cycling. To enhance user experience, I focused on improving the UI/UX design, making navigation intuitive and content easily accessible. The site also includes responsive design elements to ensure a seamless experience across different devices.

Tools/Technologies Used

This website was built using WordPress for content management and customization. I used Photoshop for image editing and optimization.
